## 🎯 **KULLANIM TALİMATLARI**

### **Hızlı Başlatma:**
```bash
# Windows'da çift tıklayın:
baslat.bat
```

### **Manuel Başlatma:**
```bash
# Terminal 1 - Backend
call .venv\Scripts\activate
python backend/app.py

# Terminal 2 - Frontend  
call .venv\Scripts\activate
cd frontend
npm start
```

### **Erişim:**
- **🖥️ Ana Uygulama:** http://localhost:3000
- **🔧 Backend API:** http://localhost:5000
